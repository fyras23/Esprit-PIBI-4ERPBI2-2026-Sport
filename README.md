# 🏀 Esprit-PIBI-4ERPBI2-2026-Sport

## 📌 Project Overview
This project is part of the **PIBI (Projet d'Intégration Business Intelligence)** at **ESPRIT Engineering School**.  
The goal of the project is to design and implement a **Business Intelligence data integration solution** for sports-related data using **Talend**.

The system collects, processes, and transforms sports data to prepare it for analysis and reporting.

---

## 🎯 Objectives
- Extract sports data from different sources
- Transform and clean the data
- Load the processed data into a data warehouse
- Prepare the data for Business Intelligence and analytics

---

## 🛠 Technologies Used
- **Talend Open Studio for Data Integration**
- **Java**
- **SQL**
- **Git & GitHub**
- **ETL Processes**

## ⚙️ ETL Workflow
The ETL workflow follows these steps:

1. **Extract**
   - Data collected from source systems

2. **Transform**
   - Data cleaning
   - Data transformation
   - Data formatting

3. **Load**
   - Data loaded into the target database for analysis

---

## 🚀 How to Run the Project

1. Open **Talend Open Studio**
2. Import the project from the repository
3. Open the **process folder**
4. Run the ETL jobs from Talend
